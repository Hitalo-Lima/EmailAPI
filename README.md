## 🚀 Tecnologias

Esse projeto está em  desenvolvimento com as seguintes tecnologias:

- Golang
- Html e Css


## 💻 Requisitos

- **Golang**: Versão 1.18 ou superior
- **Biblioteca**: github.com/go-gomail/gomail : Pacote para enviar e-mails.
